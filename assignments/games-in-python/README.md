
# 📘 Assignment: Games in Python

## 🎯 Objective

Build a simple Hangman game in Python while practicing core programming concepts like loops, conditionals, string handling, and user input validation.

## 📝 Tasks

### 🛠️ Build the Core Hangman Game

#### Description
Create a Hangman game that picks a random word and lets the player guess one letter at a time.

#### Requirements
Completed program should:

- Select a random word from a predefined list
- Display progress using underscores for unknown letters (for example: `_ _ _ _ _`)
- Accept one letter guess per turn from the user
- Track remaining incorrect attempts
- End the game when the word is fully guessed or attempts reach zero


### 🛠️ Improve Gameplay and Feedback

#### Description
Enhance the game with clear feedback and simple input checks so the player has a smooth experience.

#### Requirements
Completed program should:

- Prevent duplicate guesses from reducing attempts
- Handle invalid input (empty input, more than one character, or non-letter characters)
- Display clear win and lose messages with the final word
- Show guessed letters and attempts remaining after each turn
